ChatDev IDE是一個集成多個大語言模型的Chrome拓展，它由三部分組成：**遊戲模式**、**聊天模式**和**Prompt IDE**。

你可以個性化這些NPC，自定義位置的提示，用可視化的提示編輯器構建你的GPTs，運行GPTs讓NPCs自我協作。
